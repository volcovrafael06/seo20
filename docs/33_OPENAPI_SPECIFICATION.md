# 33 — OpenAPI Specification

Define o padrão da API REST.

## Convenções
- Prefixo: /api/v1
- JSON UTF-8
- JWT Supabase
- Paginação cursor/page
- Erros padronizados (RFC7807 futuramente)

## Recursos
- /clients
- /audits
- /reports
- /recommendations
- /integrations
- /automations
- /knowledge

## Versionamento
- v1 estável
- mudanças incompatíveis somente em v2.
