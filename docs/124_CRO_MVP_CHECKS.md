# 124 — CRO MVP Checks

## Objetivo

Definir a primeira versão dos checks de CRO para páginas de conversão analisadas pelo SEO20.

## Checks iniciais

- Proposta de valor visível.
- CTA principal presente.
- CTA acima da dobra quando aplicável.
- Formulário simples.
- Prova social visível.
- Informações de contato claras.
- Página mobile legível.
- Ausência de excesso de distrações.

## Entradas

- HTML.
- Texto principal.
- Headings.
- Links e botões.
- Formulários.
- Dados visuais futuros.

## Saída

- CRO Score.
- Problemas de conversão.
- Recomendações por prioridade.
- Sugestões de teste.

## Regras

- Não prometer aumento garantido de conversão.
- Separar hipótese de problema confirmado.
- Priorizar mudanças simples de alto impacto.

## Evolução futura

- Integração com GA4.
- Eventos de conversão.
- Heatmaps.
- Gravações de sessão.
- Testes A/B.
