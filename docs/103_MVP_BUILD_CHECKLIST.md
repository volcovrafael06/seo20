# 103 — MVP Build Checklist

## Objetivo

Listar entregáveis mínimos para validar o MVP do SEO20.

## Infraestrutura

- Repositório organizado.
- Variáveis de ambiente.
- Supabase configurado.
- Redis configurado.
- Backend rodando.
- Frontend rodando.
- Worker rodando.

## Produto

- Login.
- Organização.
- Usuários.
- Clientes.
- Sites.
- Auditoria.
- Dashboard.
- Recomendações.
- Relatório PDF.

## Segurança

- RLS inicial.
- RBAC.
- Service role apenas no backend.
- Logs básicos.
- Rate limit inicial.

## Qualidade

- Testes mínimos.
- Lint.
- Typecheck.
- Healthcheck.
- Smoke test.

## Critério de pronto

O MVP está pronto quando uma agência consegue cadastrar um cliente, executar auditoria, visualizar recomendações e gerar um relatório PDF.
