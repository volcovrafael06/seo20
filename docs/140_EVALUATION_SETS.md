# 140 — Evaluation Sets

## Objetivo

Definir conjuntos de avaliação para validar auditorias, recomendações e relatórios do SEO20.

## Tipos

- Site pequeno.
- Site com erros conhecidos.
- Landing page.
- E-commerce.
- Negócio local.
- Conteúdo informacional.

## Estrutura

Cada caso deve conter:

- id.
- descrição.
- entrada.
- saída esperada.
- critérios de aceite.

## Métricas

- Precisão.
- Cobertura.
- Clareza.
- Validade de saída.
- Custo de execução.

## Regras

- Casos devem ser versionados.
- Resultados devem ser comparáveis ao longo do tempo.
- Mudanças relevantes devem registrar motivo.

## Evolução

- Execução automática.
- Painel de qualidade.
- Comparação entre versões.
