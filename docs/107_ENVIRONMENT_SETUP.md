# 107 — Environment Setup

## Objetivo

Definir os passos para preparar ambiente local, staging e produção.

## Ambiente local

Necessário:

- Node.js.
- Python.
- Redis.
- Supabase.
- Git.

## Variáveis

As variáveis devem ser documentadas em arquivo de exemplo, sem valores reais.

## Execução

O desenvolvimento local deve permitir rodar frontend, backend e worker separadamente.

## Staging

Ambiente para validação antes de produção.

## Produção

Ambiente protegido com monitoramento, backups e controle de acesso.
