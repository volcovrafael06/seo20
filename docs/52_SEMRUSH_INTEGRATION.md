# 52 — Semrush Integration

## Objetivo

Documentar a integração com Semrush para dados de SEO, concorrência, palavras-chave, backlinks, tráfego orgânico e tráfego pago.

## Dados principais

- Palavras-chave orgânicas.
- Posição média.
- Volume.
- Dificuldade.
- CPC.
- Concorrentes orgânicos.
- Backlinks.
- Domínios de referência.
- Anúncios pagos detectados.
- Keyword Gap.
- Backlink Gap.

## Uso no SEO20

A integração alimenta:

- Keyword Engine.
- SERP Engine.
- Competitor Intelligence.
- Backlink Engine.
- Ads Intelligence.
- Report Engine.

## Processo

1. Normalizar domínio do cliente.
2. Consultar visão geral orgânica.
3. Consultar palavras orgânicas.
4. Consultar concorrentes.
5. Consultar backlinks.
6. Cruzar com dados internos.
7. Gerar recomendações priorizadas.

## Regras de cache

Dados de Semrush devem ser cacheados por auditoria para reduzir custo e evitar chamadas duplicadas.

## Saídas

- Gaps de palavras-chave.
- Oportunidades de conteúdo.
- Concorrentes reais.
- Backlinks e link gap.
- Indicadores de Ads quando disponíveis.
