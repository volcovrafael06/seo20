# 115 — Report MVP

## Objetivo

Definir o relatório mínimo do SEO20.

## Seções

- Capa.
- Resumo.
- Score geral.
- Scores por área.
- Achados principais.
- Recomendações.
- Próximos passos.

## Regras

- Relatório deve ser gerado a partir de uma auditoria.
- Arquivo deve ficar privado por padrão.
- Geração deve ocorrer em fila.
- O status deve ser consultável.

## Evolução

- White-label.
- Templates por agência.
- Portal do cliente.
- Histórico comparativo.
