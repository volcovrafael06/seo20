# 122 — Local SEO MVP

## Objetivo

Definir a primeira versão do módulo de SEO Local do SEO20.

## Fontes

- Dados cadastrados do cliente.
- Site.
- Google Business Profile quando conectado.
- Páginas locais do site.

## Checks iniciais

- Nome da empresa presente no site.
- Endereço presente quando aplicável.
- Telefone presente quando aplicável.
- Página de contato clara.
- Uso de cidade/região em páginas locais.
- Dados consistentes com cadastro.
- Presença de links para perfil local.

## Scores

- Local Presence Score.
- Contact Clarity Score.
- Location Content Score.

## Saída

- Diagnóstico local.
- Problemas encontrados.
- Recomendações.
- Prioridade.

## Evolução futura

- Google Business Profile completo.
- Reviews.
- Categorias.
- Fotos.
- Posts.
- Rankings locais.
- Map pack tracking.
