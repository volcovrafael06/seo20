# 45 — Roadmap Técnico V2/V3/V4

## Objetivo

Definir evolução do SEO20 após o MVP.

## V1 — MVP

Foco:

- Cadastro de clientes.
- Auditoria SEO básica.
- Auditoria local básica.
- Relatório PDF.
- Dashboard inicial.
- Supabase Auth.
- Workers básicos.

## V2 — Plataforma operacional

Foco:

- Keyword Engine completo.
- SERP Engine.
- Competitor Intelligence.
- Backlink Engine.
- Google Search Console.
- GA4.
- Report Engine avançado.
- Opportunity Center.

## V3 — IA e automação assistida

Foco:

- Strategy Agent completo.
- Report Agent avançado.
- Automation Agent em modo rascunho.
- WordPress Integration.
- GitHub Integration.
- Google Ads sugestões.
- Meta Ads sugestões.
- Knowledge Base/RAG.

## V4 — SaaS white-label

Foco:

- Billing.
- Planos.
- Multiempresa avançado.
- Marca branca.
- Domínio customizado.
- Limites de uso.
- Relatórios com identidade da agência.

## V5 — Enterprise

Foco:

- Data Warehouse.
- Knowledge Graph.
- Event Bus avançado.
- API pública.
- Marketplace de templates.
- Multi-LLM Router.
- Automação com rollback.

## Critério de avanço

Uma versão só deve avançar quando:

- Testes críticos passarem.
- Documentação estiver atualizada.
- Segurança revisada.
- Custos de API monitorados.
- Feedback real da agência validado.
