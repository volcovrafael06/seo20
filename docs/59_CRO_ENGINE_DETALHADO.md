# 59 — CRO Engine Detalhado

## Objetivo

O CRO Engine identifica oportunidades para aumentar conversões sem depender apenas de mais investimento em mídia.

## Entradas

- Landing pages.
- GA4.
- Microsoft Clarity.
- Heatmaps.
- Eventos.
- Funis.
- Formulários.
- Dados de campanhas.

## Áreas analisadas

### Oferta

Avaliar:

- Clareza da promessa.
- Diferencial.
- Benefício principal.
- Prova de valor.
- Correspondência com intenção do usuário.

### Above the Fold

Verificar:

- Headline.
- Subheadline.
- CTA principal.
- Imagem/hero.
- Velocidade percebida.
- Confiança inicial.

### CTA

Avaliar:

- Texto.
- Posição.
- Contraste.
- Frequência.
- Clareza.
- Próximo passo.

### Formulários

Verificar:

- Número de campos.
- Campos obrigatórios.
- Erros.
- Validação.
- Taxa de abandono.

### Prova social

Verificar:

- Depoimentos.
- Avaliações.
- Cases.
- Logos.
- Certificações.
- Antes/depois quando aplicável.

### Mobile

Avaliar:

- Usabilidade.
- Botões.
- Formulários.
- WhatsApp.
- Velocidade.
- Scroll.

## Score CRO

Pesos sugeridos:

- Oferta: 20%.
- CTA: 20%.
- Formulário: 20%.
- Velocidade: 15%.
- Mobile: 15%.
- Confiança: 10%.

## Saída

- Score CRO.
- Lista de fricções.
- Recomendações priorizadas.
- Evidências comportamentais.
- Plano de testes A/B.
- Sugestões para landing pages.

## Integração com Ads

Se campanhas pagas enviam tráfego para páginas com baixa conversão, o sistema deve marcar a oportunidade como prioridade alta.
