# 61 — Content Engine

## Objetivo

O Content Engine é responsável por diagnosticar, planejar e priorizar conteúdo orgânico para SEO, SEO Local e GEO.

## Entradas

- Crawl do site.
- Sitemap.
- Google Search Console.
- GA4.
- SERP Engine.
- Keyword Engine.
- Competitor Intelligence.
- Dados de negócio do cliente.
- Perguntas frequentes.
- Produtos e serviços.

## Responsabilidades

- Inventariar páginas existentes.
- Classificar intenção de busca.
- Identificar gaps de conteúdo.
- Identificar canibalização.
- Priorizar atualizações.
- Sugerir novas páginas.
- Sugerir clusters semânticos.
- Preparar briefings editoriais.
- Alimentar o Report Engine e Strategy Agent.

## Tipos de conteúdo

- Página institucional.
- Página de serviço.
- Página local.
- Artigo informacional.
- FAQ.
- Comparativo.
- Glossário.
- Case.
- Landing page.

## Métricas analisadas

- Impressões.
- Cliques.
- CTR.
- Posição média.
- Tráfego orgânico.
- Conversões.
- Taxa de engajamento.
- Backlinks por página.
- Cobertura semântica.
- Atualidade.
- Profundidade.

## Diagnósticos

### Conteúdo ausente

Quando concorrentes ranqueiam para termos relevantes e o cliente não possui página adequada.

### Conteúdo fraco

Quando a página existe, mas não cobre intenção, entidades, perguntas e estrutura suficientes.

### Conteúdo canibalizado

Quando múltiplas URLs competem pelo mesmo conjunto de termos.

### Conteúdo desatualizado

Quando a página perde tráfego, não possui atualização recente ou apresenta informações incompletas.

## Saídas

- Content Score.
- Mapa de páginas.
- Content gap.
- Briefings.
- Clusters.
- Priorização.
- Recomendações para GEO.
- Recomendações para linkagem interna.

## Priorização

A prioridade deve considerar:

- Potencial de tráfego.
- Potencial comercial.
- Dificuldade.
- Gap competitivo.
- Custo de produção.
- Impacto em conversão.

## Integrações

- Keyword Engine.
- SERP Engine.
- GEO Engine.
- Internal Link Engine.
- Report Engine.
- Automation Agent.

## Regras

- Não gerar conteúdo sem briefing.
- Não sugerir páginas duplicadas.
- Não priorizar volume sem intenção comercial.
- Não prometer ranqueamento garantido.
- Toda recomendação deve conter evidência.
