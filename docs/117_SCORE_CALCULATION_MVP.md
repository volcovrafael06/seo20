# 117 — Score Calculation MVP

## Objetivo

Definir a primeira versão do cálculo de scores do SEO20.

## Scores iniciais

- Score geral.
- Score técnico.
- Score de conteúdo básico.
- Score de indexabilidade.
- Score de links internos.

## Escala

Todos os scores usam escala de 0 a 100.

## Categorias

- 0 a 39: crítico.
- 40 a 69: atenção.
- 70 a 89: bom.
- 90 a 100: excelente.

## Peso inicial

Sugestão para MVP:

- Técnico: 40%.
- Conteúdo básico: 25%.
- Indexabilidade: 20%.
- Links internos: 15%.

## Regras

- Score deve ser reproduzível.
- Fórmula deve ser documentada.
- Penalidades devem considerar severidade.
- Problemas críticos devem impactar mais.
- Score não deve depender de IA no MVP.

## Evolução futura

- Pesos por segmento.
- Score histórico.
- Benchmark.
- Ajuste por impacto comercial.
- Score GEO e CRO.
