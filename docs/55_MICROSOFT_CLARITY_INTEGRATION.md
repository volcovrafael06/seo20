# 55 — Microsoft Clarity Integration

## Objetivo

Documentar a integração com Microsoft Clarity para análise comportamental, mapas de calor, gravações de sessão, rage clicks, dead clicks e oportunidades de CRO.

## Dados principais

- Heatmaps.
- Session recordings.
- Rage clicks.
- Dead clicks.
- Scroll depth.
- Device type.
- Browser.
- User journey.
- Páginas com fricção.

## Uso no SEO20

A integração alimenta:

- CRO Engine.
- Landing Page Audit.
- UX Recommendations.
- Report Engine.
- Strategy Agent.

## Processo

1. Conectar conta/projeto Clarity.
2. Selecionar site do cliente.
3. Coletar métricas por página.
4. Identificar fricções.
5. Cruzar com dados de GA4 e campanhas.
6. Gerar recomendações de CRO.

## Exemplos de recomendações

- CTA pouco clicado.
- Formulário com abandono.
- Botão não funcional.
- Conteúdo importante abaixo da dobra.
- Página mobile com fricção.

## Cuidados

- Respeitar privacidade.
- Não exportar dados pessoais desnecessários.
- Não incluir gravações sensíveis em relatórios públicos.
- Usar dados agregados sempre que possível.

## Saídas

- Score CRO.
- Evidências comportamentais.
- Prioridades de UX.
- Recomendações de landing page.
- Insights para otimização de conversão.
