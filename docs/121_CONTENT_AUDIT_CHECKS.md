# 121 — Content Audit Checks

## Objetivo

Definir os checks iniciais de conteúdo para apoiar auditorias SEO e recomendações editoriais.

## Checks MVP

- Title pouco descritivo.
- Meta description ausente.
- Conteúdo muito curto.
- H1 ausente.
- Estrutura de headings fraca.
- Falta de perguntas frequentes.
- Baixa clareza de intenção.
- Duplicidade provável entre páginas.

## Dados usados

- Title.
- Meta description.
- Headings.
- Texto principal.
- URL.
- Tipo de página.

## Classificação de intenção

Categorias iniciais:

- Informacional.
- Comercial.
- Transacional.
- Navegacional.
- Local.

## Saída

Cada página deve receber diagnóstico textual curto e recomendações de melhoria.

## Regras

- Não usar IA como única fonte de decisão crítica no MVP.
- Evidências devem estar ligadas ao conteúdo coletado.
- Recomendações devem ser práticas.

## Evolução futura

- Clusterização de tópicos.
- Briefings automáticos.
- Entidades semânticas.
- Gap de conteúdo.
- Comparação com concorrentes.
