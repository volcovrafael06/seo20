# 120 — Technical SEO Checks

## Objetivo

Definir os checks técnicos mínimos e evolutivos do SEO20.

## Checks MVP

- Status HTTP inválido.
- Redirecionamentos.
- Title ausente.
- Title duplicado.
- Meta description ausente.
- Meta description duplicada.
- H1 ausente.
- Múltiplos H1.
- Canonical ausente.
- Canonical externo.
- Noindex.
- Links internos quebrados.
- Imagens sem alt.

## Severidade

### Alta

Problemas que afetam rastreamento, indexação ou acesso à página.

### Média

Problemas que reduzem qualidade técnica ou clareza.

### Baixa

Melhorias incrementais.

## Evidência

Cada check deve produzir:

- URL afetada.
- Tipo do problema.
- Valor encontrado.
- Valor esperado quando aplicável.

## Regras

- Checks devem ser determinísticos.
- Resultado deve ser reproduzível.
- Cada achado deve estar ligado a uma auditoria.
- Problemas iguais devem ser agrupáveis.

## Evolução futura

- Core Web Vitals.
- Dados estruturados.
- Sitemap.
- Robots.
- Hreflang.
- Paginação.
- Facetas de e-commerce.
