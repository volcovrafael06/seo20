# 51 — Ahrefs Integration

## Objetivo

Documentar como o SEO20 poderá usar dados do Ahrefs para auditoria de autoridade, backlinks, palavras-chave orgânicas, páginas mais fortes e análise de concorrentes.

## Dados principais

- Domínios de referência.
- Backlinks.
- Anchor texts.
- Páginas mais linkadas.
- Links novos e perdidos.
- Autoridade do domínio.
- Autoridade de URL.
- Palavras-chave orgânicas.
- Estimativa de tráfego orgânico.
- Concorrentes orgânicos.

## Uso no SEO20

O Ahrefs alimenta principalmente:

- Backlink Engine.
- Competitor Intelligence Engine.
- Content Gap.
- Link Gap.
- Strategy Agent.
- Report Engine.

## Processo

1. Receber domínio do cliente.
2. Consultar visão geral do domínio.
3. Coletar backlinks e domínios de referência.
4. Coletar páginas mais fortes.
5. Comparar com concorrentes prioritários.
6. Gerar oportunidades de autoridade.
7. Enviar dados normalizados ao Backlink Engine.

## Cuidados

- Respeitar limites de API.
- Usar cache para evitar custo repetido.
- Não depender apenas de uma ferramenta para autoridade.
- Classificar oportunidades com cautela para evitar spam.

## Saídas esperadas

- Score de backlinks.
- Link Gap.
- Lista de domínios replicáveis.
- Riscos de toxicidade.
- Páginas com maior autoridade.
- Recomendações de Digital PR.
