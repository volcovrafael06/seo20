# 149 — Glossary

## Objetivo

Centralizar termos usados na documentação do SEO20.

## Termos

### Audit

Execução de análise de um site, página, campanha ou conjunto de dados.

### Finding

Achado detectado por uma auditoria.

### Recommendation

Ação sugerida com base em achado ou oportunidade.

### Opportunity

Possibilidade de melhoria com impacto potencial.

### Tenant

Organização isolada dentro da plataforma.

### Organization

Conta principal de uma agência ou empresa usuária.

### Client

Cliente atendido pela organização.

### Website

Site vinculado a um cliente.

### Report

Documento gerado a partir de dados e auditorias.

### RAG

Pipeline que recupera contexto relevante antes de gerar resposta com IA.

### GEO

Otimização para mecanismos generativos.

### CRO

Otimização de conversão.

### RBAC

Controle de acesso baseado em papéis.

### RLS

Políticas de segurança por linha no banco.

## Evolução

Este glossário deve ser atualizado sempre que novos termos estruturais forem adicionados.
