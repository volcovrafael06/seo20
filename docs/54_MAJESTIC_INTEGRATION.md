# 54 — Majestic Integration

## Objetivo

Documentar o uso do Majestic para análise de backlinks, Trust Flow, Citation Flow, topical relevance e histórico de links.

## Dados principais

- Trust Flow.
- Citation Flow.
- Topical Trust Flow.
- Referring domains.
- Backlinks.
- Anchor text.
- Link context.
- Histórico de links.

## Uso no SEO20

Majestic será usado como fonte complementar para:

- Backlink Engine.
- Link Risk Analysis.
- Autoridade temática.
- Link Gap.
- Benchmark com concorrentes.

## Processo

1. Consultar domínio do cliente.
2. Coletar Trust Flow e Citation Flow.
3. Coletar tópicos principais.
4. Coletar backlinks relevantes.
5. Comparar com concorrentes.
6. Classificar oportunidades e riscos.

## Regras

- Usar Topical Trust Flow para avaliar relevância temática.
- Não recomendar links apenas por métrica alta.
- Priorizar contexto, relevância e segurança.

## Saídas

- Autoridade temática.
- Link risk.
- Oportunidades editoriais.
- Comparação de confiança.
- Evidências para relatório.
