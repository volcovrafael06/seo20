# 142 — Knowledge Base Structure

## Objetivo

Definir a estrutura da base de conhecimento do SEO20 para suporte, onboarding e treinamento de usuários.

## Categorias

- Primeiros passos.
- Clientes e sites.
- Auditorias.
- Relatórios.
- Integrações.
- Recomendações.
- Portal do cliente.
- Billing.
- Permissões.
- Erros comuns.

## Formato dos artigos

- Título claro.
- Problema ou objetivo.
- Passo a passo.
- Observações.
- Limitações.
- Links relacionados.

## Regras

- Artigos devem ser curtos e práticos.
- Cada artigo deve cobrir um tema principal.
- Conteúdo deve ser revisado quando a interface mudar.
- Termos técnicos devem ter explicação simples.

## Evolução futura

- Busca interna.
- Artigos por plano.
- Sugestões automáticas.
- Vídeos curtos.
- Conteúdo white-label para agências.
