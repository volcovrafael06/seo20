# 53 — Moz Integration

## Objetivo

Documentar o uso de dados do Moz para complementar a análise de autoridade, links e competitividade orgânica.

## Dados principais

- Domain Authority.
- Page Authority.
- Linking domains.
- Inbound links.
- Spam Score.
- Anchor text.
- Top pages.

## Uso no SEO20

O Moz pode complementar dados de Ahrefs, Semrush, Majestic e DataForSEO para criar uma visão mais robusta de autoridade.

## Processo

1. Consultar domínio do cliente.
2. Consultar métricas de autoridade.
3. Consultar links principais.
4. Comparar com concorrentes.
5. Normalizar métricas no Backlink Engine.

## Normalização

Como cada ferramenta usa métricas proprietárias, o SEO20 deve converter autoridade para uma escala interna, sem apresentar uma única ferramenta como verdade absoluta.

## Saídas

- Autoridade normalizada.
- Risco de spam.
- Comparação de autoridade.
- Oportunidades de links.
- Evidências para relatório executivo.
