# 123 — GEO MVP Checks

## Objetivo

Definir a primeira versão dos checks de GEO para avaliar preparo do conteúdo para mecanismos generativos.

## Checks iniciais

- Resposta direta para perguntas importantes.
- Estrutura clara de headings.
- Conteúdo com entidades identificáveis.
- Presença de FAQ quando aplicável.
- Explicações objetivas.
- Autoridade e confiança visíveis.
- Dados estruturados básicos.

## Entradas

- Conteúdo da página.
- Headings.
- Schema detectado.
- Dados da empresa.
- Perguntas relevantes.

## Saída

- GEO Score.
- Lacunas de resposta.
- Recomendações de estrutura.
- Sugestões de FAQ.
- Sugestões de entidades.

## Regras

- Recomendações devem ser claras e executáveis.
- Não prometer presença em respostas generativas.
- Indicar incerteza quando necessário.

## Evolução futura

- Monitoramento de menções por IA.
- Comparação com concorrentes.
- Graph RAG.
- Auditoria de entidades.
- Conteúdo orientado a perguntas.
