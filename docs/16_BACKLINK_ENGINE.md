# 16 — Backlink Engine

## Objetivo

O Backlink Engine analisa autoridade, perfil de links, oportunidades e riscos. Ele deve apoiar estratégias seguras de crescimento orgânico sem incentivar spam ou criação artificial de links de baixa qualidade.

## Fontes

- Ahrefs.
- Semrush.
- Moz.
- Majestic.
- DataForSEO.
- Open PageRank.
- Google Search Console quando disponível.

## Métricas principais

- Backlinks totais.
- Domínios de referência.
- Autoridade do domínio.
- Autoridade da página.
- Follow/nofollow.
- Anchor text.
- TLDs.
- País de origem.
- Links perdidos.
- Links novos.
- Toxicidade.
- Link velocity.

## Link Gap

Comparar cliente com concorrentes prioritários.

Perguntas respondidas:

- Quais domínios linkam para concorrentes e não para o cliente?
- Quais links são replicáveis?
- Quais são editoriais?
- Quais são diretórios legítimos?
- Quais exigem relacionamento ou conteúdo?

## Classificação de oportunidades

### Alta segurança

- Associações comerciais.
- Portais locais confiáveis.
- Fornecedores.
- Parceiros.
- Clientes B2B.
- Entrevistas.
- Cases.
- Diretórios oficiais.

### Média segurança

- Guest posts relevantes.
- Conteúdos colaborativos.
- Podcasts.
- Estudos de mercado.
- Listas editoriais.

### Baixa segurança / evitar

- Diretórios genéricos em massa.
- PBNs.
- Comentários automatizados.
- Compra de links sem relevância.
- Sites sem tráfego e sem contexto.
- Anchors exatos em excesso.

## Digital PR

O sistema deve sugerir ativos que podem atrair links reais:

- Estudos locais.
- Relatórios de mercado.
- Guias completos.
- Infográficos.
- Calculadoras.
- Pesquisas com dados próprios.
- Cases com resultados.

## Monitoramento

Rotina semanal:

- Verificar novos backlinks.
- Verificar backlinks perdidos.
- Identificar links tóxicos.
- Atualizar autoridade.
- Alertar quedas bruscas.

## Score de backlinks

Pesos sugeridos:

- Domínios de referência: 25%.
- Qualidade dos domínios: 30%.
- Diversidade de anchors: 15%.
- Link velocity saudável: 10%.
- Link gap contra concorrentes: 15%.
- Toxicidade baixa: 5%.

## Regras de segurança

O sistema nunca deve recomendar automação em massa de backlinks. Toda automação deve ser limitada a cadastros legítimos, preenchimento assistido e geração de materiais para outreach humano.

## Saída no relatório

- Comparativo de autoridade.
- Link gap.
- Oportunidades priorizadas.
- Riscos.
- Plano de aquisição de links.
- Alertas de toxicidade.
- Próximas ações.
