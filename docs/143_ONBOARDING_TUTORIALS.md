# 143 — Onboarding Tutorials

## Objetivo

Definir tutoriais guiados para novos usuários do SEO20.

## Tutoriais iniciais

### Criar organização

Explica como iniciar a conta principal.

### Cadastrar cliente

Mostra como registrar um cliente e seus dados básicos.

### Cadastrar site

Mostra como adicionar o site principal do cliente.

### Rodar primeira auditoria

Mostra como iniciar e acompanhar uma auditoria.

### Gerar relatório

Mostra como criar e acessar o relatório final.

## Regras

- Tutorial deve ser curto.
- Cada etapa deve ter objetivo claro.
- Usuário deve poder pular o tutorial.
- Progresso deve ser salvo.

## Métricas

- Conclusão do onboarding.
- Tempo até primeira auditoria.
- Tempo até primeiro relatório.
- Etapas abandonadas.
