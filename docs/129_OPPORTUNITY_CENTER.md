# 129 — Opportunity Center

## Objetivo

Definir o Opportunity Center do SEO20, área responsável por consolidar oportunidades de crescimento e priorizar ações.

## Fontes

- Auditorias técnicas.
- Keywords.
- Concorrentes.
- Backlinks.
- CRO.
- Ads.
- GEO.
- Histórico do cliente.

## Tipos de oportunidade

- Correção técnica.
- Conteúdo novo.
- Otimização de página existente.
- Melhoria de conversão.
- Melhoria em campanha paga.
- Link building.
- SEO local.
- GEO.

## Priorização

Critérios:

- Impacto.
- Esforço.
- Urgência.
- Confiança.
- Dependências.
- Valor comercial.

## Saída

- Lista de oportunidades.
- Score de prioridade.
- Justificativa.
- Próximo passo.
- Módulo de origem.

## Regras

- Oportunidades duplicadas devem ser agrupadas.
- Cada oportunidade deve ter origem rastreável.
- Usuário deve poder aceitar, ignorar ou transformar em tarefa.

## Evolução futura

- Priorização por IA.
- Estimativa de receita.
- Roadmap automático.
- Integração com GitHub e WordPress.
- Alertas recorrentes.
